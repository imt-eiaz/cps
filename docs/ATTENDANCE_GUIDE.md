# Attendance System - Teacher Guide

## 📋 Overview

The Attendance System allows teachers to quickly and efficiently mark daily attendance for their classes.

## 🚀 Getting Started

### 1. Access the Attendance Page

- Click on **"Attendance"** in the sidebar menu
- The page will open with options to select a class and date

### 2. Select Your Class and Date

- **Select Class**: Choose the class you want to mark attendance for from the dropdown
  - The dropdown shows: Class Name - Section (ClassCode) - Number of students
- **Select Date**: Pick the date for which you want to mark attendance
  - You cannot mark attendance for future dates
  - Default is set to today's date

### 3. Mark Attendance

#### Individual Student Marking:

1. Find the student in the list
2. Click on the status dropdown (default: "Not Marked")
3. Select one of:
   - **Present**: Student is present
   - **Absent**: Student is absent
   - **Late**: Student arrived late
   - **Leave**: Student on authorized leave
4. (Optional) Add remarks explaining the status

#### Bulk Actions:

- **All Present**: Marks all students as present instantly
- **All Absent**: Marks all students as absent instantly
- **Clear All**: Removes all attendance marks (use if you made mistakes)

### 4. Save Attendance

**Method 1**: Click the blue **"Save Attendance"** button
**Method 2**: Press `Ctrl+S` (or `Cmd+S` on Mac)

- The system will save all attendance records
- You'll see a green success message
- The data will automatically refresh

### 5. Additional Features

#### Search Students:

- Use the search bar to filter students by:
  - First name
  - Last name
  - Admission number

#### Export Data:

- Click **"Export CSV"** to download attendance records
- File includes: Student name, admission number, status, and remarks

#### Statistics Dashboard:

View real-time statistics while marking:

- Total students
- Present count
- Absent count
- Late arrivals
- Leave requests
- Not yet marked

## ⚠️ Important Notes

1. **Unsaved Changes Warning**:
   - The system tracks if you have unsaved changes
   - You'll see an orange warning: "You have unsaved changes"
   - Save button will have a pulsing animation
   - You'll be warned if you try to leave without saving

2. **Keyboard Shortcuts**:
   - `Ctrl+S` / `Cmd+S`: Save attendance

3. **Auto-Refresh**:
   - After saving, the system automatically refreshes the attendance data
   - This ensures you see the latest saved state

4. **Status Icons**:
   - ✅ Green checkmark = Present
   - ❌ Red X = Absent
   - 🕐 Orange clock = Late
   - 📄 Blue document = Leave

## 💡 Best Practices

1. **Daily Routine**:
   - Access the attendance page at the same time each day
   - Select your class
   - Mark attendance
   - Save immediately

2. **For Late Students**:
   - Initially mark them as "Absent"
   - When they arrive, change to "Late" and add arrival time in remarks
   - Save the changes

3. **For Leave Requests**:
   - Mark as "Leave"
   - Add reason in remarks (e.g., "Medical leave", "Family emergency")

4. **Double-Check Before Saving**:
   - Use the statistics at the top to verify numbers
   - Check for "Not Marked" count - should be 0 before saving

5. **Export for Records**:
   - Export attendance at the end of each week/month
   - Keep CSV files as backup records

## 🔧 Troubleshooting

### Classes not showing?

- Ensure you have classes assigned to you in the system
- Contact your administrator if you should have classes but don't see them

### Students not appearing?

- Verify students are enrolled in the selected class
- Check that the enrollment status is "active"
- Contact administrator if students are missing

### Cannot save attendance?

- Check your internet connection
- Ensure you're still logged in (session might have expired)
- Try refreshing the page and logging in again

### Changes not saving?

- Wait for the green success message before leaving
- If you see an error, try again
- Contact administrator if problem persists

## 📞 Support

If you encounter any issues:

1. Try refreshing the page
2. Clear browser cache
3. Try a different browser
4. Contact your system administrator
